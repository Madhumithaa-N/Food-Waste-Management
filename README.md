# Food Waste Management Platform

A hackathon project aimed at reducing food waste by connecting food donors (restaurants, hotels, residents, caterers) with recipients through an efficient distribution system.

## Features

- **Donate Food**: Easy-to-use form for donors to list available food with time seals
- **Browse Listings**: View available food items with distance-based pricing
- **Smart Distribution**: Self-pickup for nearby locations, minimal delivery charges for distant ones
- **Responsive Design**: Works seamlessly on all devices
- **Vibrant UI**: Eye-catching design with orange, green, and purple gradients

## Tech Stack

- React (JSX)
- React Router for navigation
- Tailwind CSS for styling
- Lucide React for icons
- Vite for build tooling

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Run the development server:
   ```bash
   pnpm dev
   ```

3. Open your browser and navigate to the local server URL

## Project Structure

- `src/app/App.tsx` - Main application component
- `src/app/components/` - All page components (Home, DonateFood, BrowseFood, HowItWorks)
- `src/app/routes.jsx` - Route configuration
- `src/styles/` - Global styles and theme configuration
